                              使用示例

1. 启动应用：
   $ source .venv/bin/activate
   $ python run_gui.py

2. 查看关节状态：
   - 右侧仪表盘显示所有12个关节数据
   - 绿色误差值 = 正常，黄色 = 需关注，红色 = 错误

3. 追踪关节历史：
   - 点击任何关节卡片
   - 底部曲线图自动切换到该关节

4. 监控系统状态：
   - 顶部状态面板实时显示系统状态
   - 颜色指示灯快速识别问题
